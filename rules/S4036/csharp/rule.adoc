== Sensitive Code Example

----
Process p = new Process();
p.StartInfo.FileName = "binary"; // Sensitive
----

== Compliant Solution

----
Process p = new Process();
p.StartInfo.FileName = @"C:\Apps\binary.exe"; // Compliant
----
