== Noncompliant Code Example

----
if (true) {  // Noncompliant
  doSomething();
}
...
if (false) {  // Noncompliant
  doSomethingElse();
}

if (!options || options === true) { doThirdThing(); }  // Noncompliant; always true
----

== Compliant Solution

----
doSomething();

doThirdThing();
----
