== Noncompliant Code Example

With the default regular expression `+^[a-z][a-zA-Z0-9_]*$+`:

----
public void doSomething(Integer PARAM) { // Noncompliant
  Integer LOCAL; // Noncompliant
  ...
}
----

== Compliant Solution

----
public void doSomething(Integer param) { 
  Integer local;  
  ...
}
----
