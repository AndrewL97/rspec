== Noncompliant Code Example

----
SELECT HASHBYTES('SHA1', MyColumn) FROM dbo.MyTable;  
----

== Compliant Solution

----
SELECT HASHBYTES('SHA2_256', MyColumn) FROM dbo.MyTable;  
----
