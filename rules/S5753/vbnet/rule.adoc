== Sensitive Code Example

At Controller level:

----
<ValidateInput(False)>
Public Function Welcome(Name As String) As ActionResult
  ...
End Function
----

== Compliant Solution

At Controller level:

----
<ValidateInput(True)>
Public Function Welcome(Name As String) As ActionResult
  ...
End Function
----
or 

----
Public Function Welcome(Name As String) As ActionResult
  ...
End Function
----
